# Security And Privacy

PharmaPath should be used with the same care you would use for any live website or app.

## Good Habits

- Avoid posting prescription numbers, insurance information, health records, or pharmacy account details in GitHub issues.
- Use only the minimum context needed when asking for help.
- Confirm urgent medical questions with a clinician or pharmacy directly.

## Reporting A Concern

For corrections or questions, email [dylan@wlim.work](mailto:dylan@wlim.work). If the concern involves account access, payment, health, portfolio, transcript, or client information, email first so it can be handled carefully.
